Rate API API Reference

An API that allows a user to get a parking rate for a time span.

API Endpoint
http://localhost:8080/v1
Schemes: http
Version: 1.0.0

Paths

rate

Get a parking rate for a time span

GET /rate
start: string (date-time)
in query

Start of the time span (inclusive)

end: string (date-time)
in query

End of the time span (inclusive)

200 OK

syntax = "proto3"; package justinb99.sampleapi.schema; message Rate { enum Status { unavailable = 0; } oneof price_or_status { int32 price = 1; Status status = 2; } }

Response Content-Types: application/octet-stream

Schema Definitions

Rate: object

price: integer (int32)

Price of the parking rate. Only populated if a rate is available for the submitted time span

status: string unavailable

Indicates parking rate is unavailable for submitted time span. Only populated when a parking rate is unavailable

Example
{
  "price": "integer (int32)",
  "status": "string"
}
